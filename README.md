# 🎬 CineStream

A modern, responsive movie discovery application built with **React** and **Vite** that allows users to explore popular movies, search for titles, save favorites, and receive personalized movie recommendations.

## 🚀 Features

* Browse Popular Movies
* Search Movies
* Infinite Scroll
* Debounced Search (500ms)
* Favorite Movies with Local Storage
* Dedicated Favorites Page
* Smart Movie Recommendation
* Responsive Design
* Lazy Loaded Images
* Loading & Error States
* Modern Streaming Platform Inspired UI

---

## 🛠️ Tech Stack

* React
* Vite
* React Router DOM
* Axios
* TMDB API
* JavaScript (ES6+)
* HTML5
* CSS3
* Local Storage
* Intersection Observer API

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Bhope-Varun/CINESTREAM.git
```

### Navigate to the project

```bash
cd CINESTREAM
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in the project root.

```env
VITE_TMDB_API_KEY=YOUR_TMDB_API_KEY
VITE_RECOMMENDATION_API_KEY=YOUR_API_KEY
```

## 📱 Application Features

### Popular Movies

* Displays trending and popular movies.
* Responsive movie grid layout.
* Displays poster, title, release year, and rating.

### Search

* Search movies by title.
* Debounced API requests for improved performance.

### Infinite Scroll

* Automatically loads additional movies while scrolling.
* Built using the Intersection Observer API.

### Favorites

* Add and remove favorite movies.
* Favorites persist using Local Storage.

### Smart Movie Recommendation

* Enter a description of the type of movie you'd like to watch.
* The application suggests a matching movie and displays its details.

### Performance Optimizations

* Lazy-loaded images
* Debounced search
* Infinite scrolling
* Efficient rendering
* Responsive UI
